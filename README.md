## ✨ Features

| Section | Description |
|---|---|
| 🗂️ **Layer Explorer** | Click any of the 7 OSI layers for in-depth info on protocols, devices, functions & threats |
| 🧠 **Mind Map** | Interactive canvas mind map — drag, zoom, and click nodes to highlight layers |
| 📊 **Comparison View** | Side-by-side layer cards with search/filter — great for quick reference |
| 📦 **Encapsulation** | Visual step-by-step breakdown of how data travels down the OSI stack |
| 💀 **Hacker Hub** | Threat actor profiles, attack chains by layer, offensive tools deep-dives |
| 🧩 **Quiz** | 12-question randomized quiz with instant feedback and score tracking |

## 📚 What's Covered

### 7 OSI Layers
- **Layer 7 — Application:** HTTP/HTTPS, FTP, SMTP, DNS, SSH
- **Layer 6 — Presentation:** TLS/SSL, AES-256, RSA, X.509 Certificates
- **Layer 5 — Session:** Kerberos, RADIUS, Session Tokens, CHAP
- **Layer 4 — Transport:** TCP, UDP, QUIC, SYN Cookies, Port Filtering
- **Layer 3 — Network:** IPv4, IPv6, BGP, OSPF, IPSec, ACLs
- **Layer 2 — Data Link:** Ethernet, Wi-Fi 802.11, ARP, VLANs, 802.1X
- **Layer 1 — Physical:** Hubs, NICs, Physical Access Control

### ⚔️ Attacks Covered
`SQL Injection` `DNS Spoofing` `ARP Spoofing` `BGP Hijacking` `SYN Flood` `MAC Flooding` `VLAN Hopping` `SSL Stripping` `HEARTBLEED` `POODLE` `Pass-the-Hash` `Kerberoasting` `NTLM Relay` `Evil Twin` `Deauth Attack` and more

### 🛠️ Tools Covered
`Nmap` `Wireshark` `Metasploit` `Bettercap` `Burp Suite` `Mimikatz` `Aircrack-ng` `SQLMap` `Hashcat` `hping3` `Scapy` `Yersinia` `Responder` `ExaBGP` and more

### 👤 Threat Actor Profiles
`Script Kiddie` `Penetration Tester` `APT Group` `Insider Threat` `Hacktivist` `Cybercriminal`

## 🚀 Getting Started

### Run Locally
No build tools or dependencies needed — it's a single HTML file.

```bash
git clone https://github.com/muhammadalihasnaat07-maker/osi-model-interactive.git
cd osi-model-interactive
# Open index.html in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Or just visit the live site:
**[https://muhammadalihasnaat07-maker.github.io/osi-model-interactive/](https://muhammadalihasnaat07-maker.github.io/osi-model-interactive/)**
